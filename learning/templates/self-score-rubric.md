# Self-Score Rubric

Score each category 1-5.

- Clarity: can I explain without code open?
- Correctness: are my statements technically accurate?
- Tradeoffs: do I name downside + mitigation?
- Debugging: do I use hypothesis-first method?
- Ownership: can I defend why this was built this way?

Target before interview: average >= 4.0 across 3 consecutive sessions.
